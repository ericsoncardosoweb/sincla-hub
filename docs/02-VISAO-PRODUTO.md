# Visao de Produto - Sincla Hub

> Posicionamento estrategico, proposta de valor e diretrizes de comunicacao

---

## 1. O Que e o Sincla Hub

### Definicao
Sincla Hub e o **portal central** do ecossistema Sincla, reunindo 6 plataformas SaaS integradas em uma experiencia unificada. Similar ao modelo Atlassian (Jira, Trello, Confluence), porem com identidade propria e foco no mercado brasileiro.

### Analogia de Mercado
```
Atlassian       = Sincla
-----------       ------
Jira            = Sincla RH (gestao de pessoas)
Trello          = Sincla Agenda (produtividade)
Confluence      = Sincla Intranet (comunicacao)
Loom            = Sincla EAD (educacao)
---             = Sincla Bolso (financas pessoais)
---             = Sincla Leads (conversao)
```

### Diferencial Competitivo
1. **Integracao nativa** entre todas as plataformas
2. **I.A. presente** em todas as ferramentas
3. **Experiencia unificada** (SSO, painel central, billing unico)
4. **Descontos cruzados** para usuarios de multiplas plataformas

## 2. Proposta de Valor

### Para Empresas (B2B)
> "Centralize a gestao de pessoas, treinamentos e comunicacao interna em um unico ecossistema integrado. Menos ferramentas, mais produtividade."

**Dores resolvidas:**
- Ferramentas desconectadas (Slack + Notion + LMS separados)
- Multiplos logins e senhas
- Faturamento fragmentado
- Dados nao integrados entre sistemas

**Beneficios:**
- Um login para tudo (SSO)
- Painel administrativo centralizado
- Relatorios unificados de engajamento
- Descontos de 50% em ferramentas pessoais para colaboradores

### Para Profissionais (B2C)
> "Suas ferramentas pessoais de produtividade, financas e aprendizado em um so lugar. Simples, integradas e com I.A."

**Dores resolvidas:**
- Apps fragmentados para cada funcao
- Falta de integracao (agenda nao fala com financas)
- Experiencias inconsistentes

**Beneficios:**
- Interface familiar entre todos os produtos
- Dados sincronizados (ex: compromisso + gasto relacionado)
- Assistente I.A. Agata em multiplas plataformas

## 3. Posicionamento de Marca

### Tom de Voz
| Caracteristica | Como aplicar |
|---------------|--------------|
| **Confiavel** | Sem hype, dados reais, SLAs claros |
| **Profissional** | Linguagem clara, sem girias |
| **Tecnologico** | Mencionar I.A. e integracao sem ser generico |
| **Acessivel** | Explicar beneficios, nao features |

### Palavras-Chave Estrategicas
**Usar:**
- Integrado, unificado, centralizado
- Inteligente, automatizado, I.A.
- Simples, pratico, direto
- Seguro, confiavel, enterprise

**Evitar:**
- Revolucionario, disruptivo (buzzwords vazias)
- O melhor, o mais (superlativos nao comprovados)
- Facil (subjetivo, preferir "em poucos cliques")

### Mensagens por Contexto

**Hero (primeira impressao):**
```
"Um hub. Todas as suas ferramentas."
-> Claro, direto, comunica unificacao
```

**Secao Produtos:**
```
"Ferramentas que transformam seu negocio"
-> Foco em resultado, nao em tecnologia
```

**Secao Enterprise:**
```
"Potencialize sua empresa"
-> Verbo de acao + beneficio claro
```

## 5. Jornadas de Usuario

### Jornada 1: Visitante -> Lead (B2C)
```
1. Chega pelo Google ou indicacao
2. Ve Hero -> entende proposta
3. Scroll para Produtos -> identifica necessidade
4. Clica "Conhecer" no produto X
5. E redirecionado para plataforma especifica
6. Faz trial ou se cadastra
```

### Jornada 2: Visitante -> Lead (B2B)
```
1. Chega pelo Google ("software RH")
2. Ve Hero -> entende que e hub
3. Scroll para Empresas -> ve beneficios B2B
4. Clica "Falar com Consultor"
5. Preenche formulario
6. Recebe contato em 24h
```

### Jornada 3: Usuario -> Cross-sell
```
1. Ja usa Sincla RH (empresa)
2. Recebe email sobre Sincla Bolso (50% off)
3. Acessa Hub pelo email
4. Ve card do Bolso com desconto
5. Ativa conta com SSO
```

## 7. Persona de Comunicacao

### Persona Primaria: Gestor de RH
- **Cargo:** Coordenador/Gerente de RH
- **Empresa:** PME (50-500 colaboradores)
- **Dor:** Ferramentas fragmentadas, planilhas, retrabalho
- **Desejo:** Sistema unico que integre tudo
- **Objecao:** "Vai ser dificil migrar"

### Persona Secundaria: Profissional Autonomo
- **Perfil:** Freelancer, consultor, autonomo
- **Dor:** Desorganizacao financeira e de agenda
- **Desejo:** Controle simples, sem complexidade
- **Objecao:** "Ja uso Excel/planilhas"

### Persona Terciaria: Decisor C-Level
- **Cargo:** CEO/CFO de PME
- **Dor:** Custos com multiplos softwares
- **Desejo:** Reducao de custo, visao consolidada
- **Objecao:** "Preciso ver ROI claro"

## 8. Diferenciais para Comunicar

### No Hero
1. **6 plataformas integradas** (nao generico)
2. **10k+ usuarios** (social proof)
3. **99.9% uptime** (confiabilidade)

### Na Secao Produtos
1. **I.A. em todas** (diferencial tecnologico)
2. **Preco competitivo** (se aplicavel)
3. **Sem codigo** (para Leads)

### Na Secao Enterprise
1. **50% desconto** para colaboradores (unico)
2. **SSO nativo** (enterprise-grade)
3. **Suporte 24/7** (confianca)

---

## 9. Anti-Padroes a Evitar

### Na Comunicacao
- "Revolucionando o mercado" -> Vazio
- "Melhor software do Brasil" -> Nao comprova
- "Inteligencia artificial de ponta" -> Generico

### No Design
- Animacoes de IA generica (ondas, particulas sem proposito)
- Gradientes em tudo (polui visual)
- Excesso de badges e CTAs (confunde)

### Na UX
- Popup de newsletter no primeiro segundo
- Chatbot invasivo
- Autoplay de video

## 10. Checklist de Qualidade

Antes de publicar qualquer alteracao, validar:

- [ ] A mensagem responde "por que Sincla?"
- [ ] O beneficio esta claro em 5 segundos
- [ ] O CTA tem destino definido
- [ ] Nao ha buzzwords vazias
- [ ] A animacao tem proposito (guiar, enfatizar, transicionar)
- [ ] Funciona em mobile
- [ ] Acessivel (contraste, foco, alt text)
